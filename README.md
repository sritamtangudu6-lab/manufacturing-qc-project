# Manufacturing Process Optimization using Quality Control Tools

## 📌 Introduction
In modern manufacturing systems, controlling conversion cost is critical for maintaining profitability and efficiency. This project demonstrates how classical quality control tools can be applied to analyze process data, detect inefficiencies, and identify root causes of excessive cost.

---

## 🎯 Problem Statement
Manufacturing processes often suffer from:
- High cycle time variation
- Machine downtime
- Material wastage
- Process instability

The objective of this project is to analyze such issues and provide data-driven insights for cost reduction.

---

## ⚙️ Approach

This project integrates three major quality engineering techniques:

### 1. Pareto Analysis
- Based on the **Pareto Principle (80/20 rule)**
- Ranks cost contributors in descending order
- Highlights critical few factors affecting overall cost

---

### 2. Statistical Process Control (SPC)

#### X-bar Chart
- Monitors subgroup averages
- Detects process mean shifts

#### R Chart
- Monitors variability within subgroups
- Detects inconsistencies in production

**Control Limits Used:**
- Upper Control Limit (UCL)
- Lower Control Limit (LCL)

---

### 3. Fishbone (Cause-and-Effect) Diagram
Provides structured root cause analysis under six major categories:
- Man
- Machine
- Method
- Material
- Measurement
- Environment

---

## 🧪 Data Description
- Synthetic dataset generated using Python
- Simulates cycle time behavior in manufacturing
- Includes random noise and abnormal spikes to mimic real-world conditions
